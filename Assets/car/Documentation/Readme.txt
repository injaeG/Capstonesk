This Asset includes:

	-A demo scene for visual showcasing

	-Fully featured and functional vehicle
	-Doors, boot, and bonnet functional
	-Rigged suspension and engine (Engine Block, Pulleys, Exhaust tip and Fan)
	-Detailed and animatable interior
	-Detailed engine bay
	-Detailed undercarriage
	-4 LOD States
	-Separate windshield, rear window, and door windows
	-Separate windshield wipers
	-Separate left and right indicator light Materials
	-Separate rear view mirror and door mirror Materials
	-Emissive map for dashboard dials
	-Normal maps for all lights included
	-Light materials already setup with emission
	-Simplified collision mesh
